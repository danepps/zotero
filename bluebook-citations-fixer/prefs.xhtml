Bluebook Citations Fixer only rewrites citations when the document's active citation style matches the style ID below. Leave it set to your Bluebook style so the plugin stays dormant under every other style. Clear the box to apply the rules under all styles. Controls when a short-form "[hereinafter ...]" tag is added to repeated same-author sources. Sources that first appear together in the same footnote always receive hereinafter treatment; the options below govern sources that first appear in different footnotes.
